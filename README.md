### Make it your own

1- Clone this project,

```
git clone https://github.com/websemantics/the-ten-commandments-for-exceptional-developers
```

2- Run npm install,

```
npm install
```

3- To see the result, browse to,

```
dist/index.html
```

4- Make change to `smart.json` then run,

```
gulp
```

5- Rinse and repeat #3
